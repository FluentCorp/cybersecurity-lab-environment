# cybersecurity-lab-environment
## Overview
Designed and deployed a multi-machine virtual lab environment using VirtualBox to simulate real-world enterprise network conditions. This project focuses on vulnerability assessment, network scanning, and threat analysis.

## Environment Setup
- Kali Linux (attacker machine)
- Windows 10 / Windows Server (target systems)
- Ubuntu Linux
- Metasploitable 2
- pfSense Firewall

## Network Configuration
- Configured NAT and Host-Only networks
- Simulated segmented enterprise network structure

## Tools Used
- Nmap (network scanning & enumeration)
- Nessus (vulnerability scanning)
- Kali Linux tools
- Wireshark (basic analysis)

## Key Activities
- Performed network discovery and port scanning using Nmap
- Identified vulnerabilities using Nessus
- Analyzed open ports, services, and system weaknesses
- Practiced exploitation in controlled environments (Metasploitable, Kioptrix)
- Documented findings and proposed mitigation strategies

## Key Takeaways
- Hands-on experience in vulnerability assessment
- Understanding of network segmentation and attack surfaces
- Improved ability to analyze and interpret security risks
